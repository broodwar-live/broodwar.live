## Overview
After 8 years without updates, Blizzard released Patch 1.18 in preparation for StarCraft: Remastered. This was a landmark moment for the BW community.

## General
- Windowed mode support
- Modern resolution support (up to 4K with Remastered)
- Updated Battle.net integration
- Removed CD key requirement — game fully free
- Support for modern operating systems (Windows 7/8/10, macOS)
- Improved anti-cheat

## Network
- New matchmaking system
- Updated server infrastructure
- Cloud save for profiles and hotkeys

## Competitive Impact
This patch brought BW into the modern era technically while preserving gameplay. The Korean ASL scene was already running on unofficial patches, but 1.18 unified the client. Together with Remastered (released August 2017), this sparked a major BW revival.
